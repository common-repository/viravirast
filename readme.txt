Viravirast is a word spellchecker / grammar correction plugin powered with AI

== Description ==
Viravirast - the Persian perfectionist for your text.

== Installation ==
1. Upload `viravirast.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Login/Register on plugin

== Frequently Asked Questions ==

= I don't have an account! How can I login? =

Just go the sign up tab and create an account through our website

== Disclaimer ==
This plugin uses an external API [here](https://viravirast.com/wp-api-doc) is the doc.

This plugin will send text from Gutenberg and classic wordpress editor when the user stops typing or double clicks on the text and it could be disabled.
